## 📂 Flagship Projects

### [Nestify | The Student Living OS](https://github.com/Sunilkumar-T/nestify)
*   **The Problem:** Fragmented, paper-heavy manual onboarding in private hostels.
*   **The Solution:** A full-stack SaaS that automates 100% of documentation, rent collection, and maintenance ticketing.
*   **Stack:** React, Supabase (Realtime), Tailwind CSS.

### [LoadLink | Logistics Optimization](https://github.com/Sunilkumar-T/loadlink)
*   **The Problem:** 30-40% of Indian trucks return empty, increasing costs and carbon footprint.
*   **The Solution:** A B2B marketplace matching return-trip capacity with demand using Govt-level ULIP and FASTag data integration.
*   **Stack:** Node.js Microservices, PostgreSQL, Google Maps Fleet API.
